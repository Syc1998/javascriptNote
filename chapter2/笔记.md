DOM document object  model 文档对象模型
===
`node`           节点  
`element`      元素  
`tag`              标签  
`mouseover`  鼠标经过  
`mouseout`   鼠标移出  
`keypress`      键盘敲击  
### dom获取页面元素 
dom在加载时是按由上至下的顺序执行
* 1,get element by id（）
* 2.get elements by name（）
* 3.get elements by tag name（）
* 4.get elements by classname（）   
dom几种常用方式
* 1。使用指定的id属性
* 2。使用指定的name属性
* 3。使用指定的标签名字 
* 4。使用指定的css类
